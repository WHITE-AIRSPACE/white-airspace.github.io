需求介绍：现有一个内嵌在APP端的H5支付页面，需要在当前页面禁止截屏以及录屏。

实现步骤：

  1. APP端监听用户截屏录屏行为
  2. APP端通知H5页面
  3. H5页面监听是否有APP端的通知

以上为实现的死了，主要运用到的知识点为， 原声APP跟JS的交互
APP端调用创建自定义事件接口 CustomEvent

关键代码实现：


[slef nativeEvaluateJavaScriptWithCallBackId:@"notify",type:'截屏状态']

App断监听到事件，并且带上数据给我们通知以后，我们就只需要在页面监听 “notify” 这个自定义事件就行

关键代码：

componentDidMount() {
      var _self = this
      _self.globalNotify = (ev) => {
          if (window.userPhoneModel !== 'Android') {
              const detail = ev.detail || {};
              const type = detail.type || '';
              console.log(type)
              if (type === 'screen_recording' || type === 'screen_capture') {
                  _self.setState({
                      safeMode: true // 页面展示安全提示
                  })
              }
          }
      }
      window.addEventListener('notify', _self.globalNotify)
  }


  componentWillUnmount() {
      window.removeEventListener('notify', this.globalNotify)
  }
